1.What is the difference between Named Export, Default export and * as export ?
Ans: Named exports allow us to export multiple values (functions , variables, classes) from a single file.
     A file can have only one default export.It is often used for the "main" thing that a file represent.
     Namespace Export(*) is used to handle everything in a module at once.This is often called Namesspace Import Or Aggregate Export.

2.What is the importance of config.js file ?
Ans: It is dedicated file used to store and manage settings, constants, and enviroment-specific variables.

3.What are React Hooks?
Ans: React Hooks are functions that let us use React features (like state and lifecycle behavior) inside functional components, without writing class components.

4.Why do we need a useState Hook ?
Ans: React's useState Hook  lets a functional component store that can change over time and trigger UI updates when it changes.
