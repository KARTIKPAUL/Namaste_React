1.What is JSX ?
Ans : JSX stands for JavaScript XML. It is a syntax extension for JavaScript often used with React to describe what the user interface should look like.

2.SuperPower of JSX ?
Ans : The "superpower" of JSX is not just that it looks like HTML; It's that it blurs the line between UI logic and UI structure. In tradition web development, we often have to jump back and forth between HTML file and a JavaScript file to make things happen. 

3.Role of type attribute in script tag? What options can I use there?
Ans : In modern web development, the type attribute on a script tag tells the browser how to interpret the contents of the script.
The type attribute determines wheather the browser treats the code as a tradoitional classic script or a JavaScript Module(ESM). This changes how variables are scoped and how files are loaded.
